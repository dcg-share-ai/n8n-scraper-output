## Requesting a BoB Fresh Card

To request a BoB Fresh Card, you need to provide the following information:

*   **Location Details:**
    *   Casa
    *   Region
    *   Address
*   **Personal Details:**
    *   Embossing Name
    *   First Name
    *   Father Name
    *   Mother Name
    *   Last Name
    *   Date Of Birth
    *   Mobile Number
    *   Nationality
    *   Email Address
*   **Identification Details:**
    *   ID Number
    *   ID Issuer
    *   ID Type
*   **Document Uploads:**
    *   Passport
    *   Front ID
    *   Back ID
    *   *Maximum allowed file size for each file is 2.5MB.*
