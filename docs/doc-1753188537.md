## Requesting a BoB Fresh Card

To request a BoB Fresh Card, you will need to provide the following information and documents:

**Personal Information:**
*   Casa
*   Region
*   Address
*   Embossing Name
*   First Name
*   Father Name
*   Mother Name
*   Last Name
*   Date Of Birth
*   Mobile Number
*   Nationality
*   ID Number
*   Email Address
*   ID Issuer
*   ID Type

**Document Uploads:**
*   Passport
*   Front ID
*   Back ID
    *   Maximum allowed file size for each file is 2.5MB.
